HTML

  • The head element contains information about the webpage
  • The body elemnet represents the visible content shown to the user
  • The head element is the parent element to the Meta element
  • The meta elemnt describes the data of the webpage not visible to the user
  • The title element is import for search engine optimization "SEO"

CSS

  • Casacading Style Sheets - CSS
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The CSS box model create a visible representation of Margins and Paddings
  • External CSS style sheets are prefered to seperate style content so it is easier to manipulate. this brings up the idea of seperation of concerns: the idea of dividing content, objects or procedures of code to easily keep track of seperate functions.
  • Adding a class selector adds CSS to a specific element in the file. You must add a [.] to select any element assigned to this class.

Git

  • Git is a version control on every computer
  • We use github as an online version control software that utilizes cloud base storage
  • Git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add all the code in the branch
  • git commit -m: this command commits the code to the branch
  • git pull origin main: this checks if the local branch is in sync with the main branch if if successful then we can git push origin
  • git push origin branch-name: this will submit the push request to the git repository

JavaScript

  • Remeber control flow executes code top to bottom
  • Array: a list of objects, numbers, or strings listed in an index
  • For loops can run a piece of code for a specified length - arrayname.length
  • Math.random can can generate a random number, this function is part of a number of javascript math functions
  • A variable is a named container that allows us to store date in our code
  • Control flow is the order in which a computer executes code in a script
